# HubFlow
HubFlow — это платформа для управления когор­тами и командами, которая помогает отслеживать прогресс участников, проводить еженедельные чекпоинты, организовывать работу менторов . Система включает дашборды прогресса, расписание встреч, уведомления и Demo Day с голосованием и презентациями проектов.
